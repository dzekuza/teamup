  <action>
Create `mobile/teamup/components/ui/bottom-sheet.tsx` — a custom bottom sheet using React Native's `Modal` with `transparent={true}` and Reanimated 4 for animation. Do NOT use `@gorhom/bottom-sheet` (broken on Expo 54 + Reanimated 4).

**Props interface `BottomSheetProps`:**
- `visible: boolean` — controls sheet visibility
- `onDismiss: () => void` — called when backdrop tapped or sheet dismissed
- `children: React.ReactNode` — sheet content
- `snapHeight?: number` — sheet height, defaults to `300`

**Internal state:**
- `const [modalVisible, setModalVisible] = useState(false)` — separate from `visible` prop to delay unmount until exit animation completes (Pitfall 3 from research)
- When `visible` becomes true: `setModalVisible(true)`, then animate in
- When `visible` becomes false: animate out, then use `runOnJS(setModalVisible)(false)` from animation completion callback

**Animation (Reanimated 4):**
- `const translateY = useSharedValue(snapHeight)` — starts offscreen
- `const backdropOpacity = useSharedValue(0)`
- `useEffect` watching `visible`:
  - If `visible=true`: set `modalVisible(true)`, then `translateY.value = withSpring(0, { damping: 20, stiffness: 200 })`, `backdropOpacity.value = withTiming(1, { duration: 250 })`
  - If `visible=false`: `translateY.value = withSpring(snapHeight, { damping: 20, stiffness: 200 }, (finished) => { if (finished) runOnJS(setModalVisible)(false); })`, `backdropOpacity.value = withTiming(0, { duration: 200 })`
- Import `runOnJS` from `react-native-reanimated` (still available in Reanimated 4 compat layer per research)

**Safe area (MOBILE-07):**
- Import `useSafeAreaInsets` from `react-native-safe-area-context`
- `const { bottom } = useSafeAreaInsets()`
- Add `bottom` to the sheet's `paddingBottom` (base 16 + insets.bottom)

**Layout structure:**
```
<Modal transparent visible={modalVisible} animationType="none" onRequestClose={onDismiss}>
  <Animated.View style={[absoluteFill, backdrop, backdropAnimStyle]}>
    <Pressable style={absoluteFill} onPress={handleDismiss} />
  </Animated.View>
  <Animated.View style={[sheet, { height: snapHeight, paddingBottom: 16 + bottom }, sheetAnimStyle]}>
    <View style={handleContainer}>
      <View style={handle} />  {/* 40x4, borderRadius 2, backgroundColor: Brand.accent */}
    </View>
    {children}
  </Animated.View>
</Modal>
```

**Styles (use Brand tokens from `@/constants/theme`):**
- Backdrop: `backgroundColor: 'rgba(0,0,0,0.6)'`
- Sheet: `position: 'absolute'`, `bottom: 0`, `left: 0`, `right: 0`, `backgroundColor: Brand.surface`, `borderTopLeftRadius: 20`, `borderTopRightRadius: 20`
- Handle container: `alignItems: 'center'`, `paddingVertical: 12`
- Handle: `width: 40`, `height: 4`, `borderRadius: 2`, `backgroundColor: Brand.accent`

**Dismiss handler:**
- `handleDismiss` function animates out first, THEN calls `onDismiss` prop via `runOnJS`

**Accessibility:**
- Backdrop `Pressable` has `accessibilityLabel="Close"` and `accessibilityRole="button"`

Named export: `export function BottomSheet`

**Important:** Use `Brand` tokens from `@/constants/theme` for surface/accent colors where applicable. The handle color `Brand.accent` and sheet background `Brand.surface`.

import { KeyboardProvider } from 'react-native-keyboard-controller';

export default function RootLayout() {

      <ThemeProvider value={colorScheme === 'dark' ? DarkTheme : DefaultTheme}>
        <Stack>
          <Stack.Screen name="(tabs)" options={{ headerShown: false }} />
          <Stack.Screen name="modal" options={{ presentation: 'modal', title: 'Modal' }} />
