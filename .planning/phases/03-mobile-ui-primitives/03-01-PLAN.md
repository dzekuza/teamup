  <name>Task 2: Create AppButton with primary/secondary variants, loading, and haptics</name>
  <files>

  <action>
Create `mobile/teamup/components/ui/button.tsx` implementing both primary and secondary button variants in a single component.

**Props interface `AppButtonProps`:**
- `label: string` — button text
- `onPress: () => void` — press handler
- `variant?: 'primary' | 'secondary'` — defaults to `'primary'`
- `loading?: boolean` — defaults to `false`
- `disabled?: boolean` — defaults to `false`

**Animation (Reanimated 4):**
- Import from `react-native-reanimated`: `useSharedValue`, `useAnimatedStyle`, `withSpring`, `Animated`
- `const scale = useSharedValue(1)`
- `animatedStyle = useAnimatedStyle(() => ({ transform: [{ scale: scale.value }] }))`
- `handlePressIn`: `scale.value = withSpring(0.96, { damping: 15, stiffness: 300 })` — only when NOT disabled and NOT loading
- `handlePressOut`: `scale.value = withSpring(1, { damping: 15, stiffness: 300 })`

**Haptics:**
- Import `* as Haptics from 'expo-haptics'`
- Call `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light)` inside `handlePressIn` (JS thread, NOT inside useAnimatedStyle)

**Layout:**
- Outer: `Pressable` with `disabled={disabled || loading}`, `accessibilityRole="button"`, `accessibilityLabel={label}`, `accessibilityState={{ disabled: disabled || loading, busy: loading }}`, `style={{ minHeight: 44, minWidth: 44 }}`
- Inner: `Animated.View` with animated style applied + variant-specific styles
- Primary: `backgroundColor: Brand.accent`, label color `Brand.accentDark`
- Secondary: `backgroundColor: 'transparent'`, `borderWidth: 1.5`, `borderColor: Brand.accent`, label color `Brand.accent`
- Disabled/loading: `opacity: 0.5`
- Button padding: `paddingVertical: 14`, `paddingHorizontal: 24`, `borderRadius: 12`, `minHeight: 52`, `alignItems: 'center'`, `justifyContent: 'center'`

**Loading state:**
- When `loading=true`, render `<ActivityIndicator color={isPrimary ? Brand.accentDark : Brand.accent} />` instead of label text

**Label text:** `fontSize: 16`, `fontWeight: '600'`

Named export: `export function AppButton`

Use `Brand` tokens from `@/constants/theme` — NOT hardcoded hex values.
